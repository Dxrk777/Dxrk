# Guía de Contribución a DXRK

## Flujo de Contribución

### 1. Fork del Repositorio
Haz fork de este repo en tu cuenta de GitHub.

### 2. Crear una Rama
```bash
git checkout -b tu-rama-caracteristica
```

### 3. Realizar Cambios
Haz tus modificaciones, agrega tests si es necesario.

### 4. Enviar Pull Request
- Push a tu fork
- Abre un PR hacia `Dxrk:main` o `Dxrk:master`
- Describe tus cambios detalladamente

### 5. Revisión
- El propietario revisará tu PR
- Podrá solicitar cambios, hacer comentarios o aprobar
- **No se hará merge sin aprobación explícita**

---

## Requisitos

- Código limpio y documentado
- Tests para nuevas funcionalidades
- Respeto a la licencia propietaria
